個人簡介

我是陳昱中，喜歡解決各種問題，興趣是各式運動。畢業於東華大學諮商與臨床心理學系，畢業後任職台北醫學大學附設醫院復健科研究助理，主要是做一些高科技新型復健的研究，目前已經離職想跨入User experience的工作領域， 它是一門很廣的領域，包含資訊、設計和心理，就我知道在台灣多半在網站和APP設計上比較有此領域的職缺，故申請這個軟體工程師扶持計畫，想要培養自己前端軟體的硬實力。

為什麼想成為工程師 ( 請以自我期許帶入：前端、後端、或全端工程師 )

因為想從前端工程師的角度進入User experience的領域，自己學習好軟體這一塊更能了解網站與App設計的困難，使自己的技能樹成長更有基底。 一方面也覺得軟體這一塊是未來的趨勢，學好軟體未來發展性更好，技術在身上不怕沒飯吃的想法，故想成為前端工程師慢慢前進。

為了成為工程師，曾經做過什麼努力？

1.從一些網頁上大致了解HTML、CSS語法，例如w3schools.com、medium。

2.翻閱書籍，例如學好跨平台網頁設計、UX從新手開始使用者體驗的100堂必修課。

3.觀看youtube 上的教學影片，詢問認識的工程師朋友。

4.平時練習撰寫一些簡單的語法

對於工程師的職業想像是什麼？

1.上班時坐在電腦前面打著程式碼修改或更新系統，遇到不會跟改或忘記的地方有個自己的可以搜尋的資料庫

2.跟公司的PM或是UI/UX設計師討論要如何編碼

3.要選在一個較少人使用系統的時間把更新好的系統上線，並測試完畢。

如果參與這個計畫，會怎麼安排學習時間？

目前待業希望利用星期一到五的時間，每天安排10小時學習當周訓練營目標，並解決當周的作業。星期六、 日安排約6小時檢視當周所學哪裡有問題，並且複習不熟的地方，期望能達到一個星期約60小時的學習時間。

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

害怕撰寫程式碼部分過慢，拖累到自己的學習進度，除了熟悉程式碼外，還要訓練自己英打的速度， 目前已經努力在訓練這一部分了。等計畫開始後也會花額外時間讓自己更熟悉打鍵盤的速度。

想要對我們說的事情？

彭彭老師你好，從三、四年前就開始關注老師的頻道，從影片中能夠清楚明白了解到軟體的相關資訊，只是覺得自己沒能夠系統化這些知識，自己都是很零碎的學習， 剛好很高興有機會剛好看到這個軟體工程師扶持計畫，有機會成為彭彭老師的學生，給自己一個很好學習的機會，還請多多指教。
